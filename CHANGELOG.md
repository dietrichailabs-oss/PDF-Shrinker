# Changelog

## 1.0.1 — 2026-07-15

### Added

- Large, obvious **RUN COMPRESSION** controls
- Desktop and Start Menu shortcuts in the Windows installer
- `Desktop\Marks Apps\PDF Shrinker` installation and release structure
- Portable EXE and coworker installer packaging

- Reproducible icon generator

- Extreme raster compression
- Ghostscript `/screen` compression
- Ghostscript `/ebook` compression
- Lossless PDF cleanup
